Mic vs Voice Module

Quick comparison for the voice-controlled light project.

More setup
Option A: Mic + External Processing + ESP32
Speak → Mic → Laptop/Cloud/Pi → Sends 1/0 to ESP32 → Light ON/OFF
Parts & Cost (₹)
  • Mic module 80 – 200
  • ESP32 board 250 – 400
  • Relay (if bulb) 80 – 200
  • LED/Bulb + holder 10 – 200
  • Wires/Breadboard 100 – 200
  • Laptop/Pi for speech Existing / 3000+
Key Points
  • Mic doesn’t understand speech
  • Needs external device + maybe internet
  • More coding and setup
  • “Advanced” but risky for demo
Total cost ≈ ₹800 – ₹1,000 (excluding laptop/Pi)
Simpler
Option B: Voice Module + ESP32
Speak → Module recognizes ON/OFF → Sends to ESP32 → Light ON/OFF
Parts & Cost (₹)
  • Voice module 1,400 – 2,200
  • ESP32 board 250 – 400
  • Relay (if bulb) 80 – 200
  • LED/Bulb + holder 10 – 200
  • Wires/Breadboard 100 – 200
Key Points
  • Understands trained commands
  • No internet or laptop needed
  • Simple wiring + code
  • More reliable for demo
Total cost ≈ ₹2,000 – ₹2,500

Quick Comparison
