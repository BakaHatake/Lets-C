Mic vs Voice Module

Comparison for the voice-controlled light project.

More setup
Option A: Mic + External Processing + ESP32
Speak → Mic → Laptop/Cloud/Pi → Sends 1/0 to ESP32 → Light ON/OFF
Brain Location:
Laptop / Cloud / Raspberry Pi does speech recognition.
Output:
LED (low voltage) or real bulb (with relay).
Parts & Cost (₹)
  • Mic module 80 – 200
  • ESP32 board 250 – 400
  • Relay (if bulb) 80 – 200
  • LED/Bulb + holder 10 – 200
  • Wires/Breadboard 100 – 200
  • Laptop/Pi for speech Existing / 3000+
Key Points
  • Mic only hears sound, doesn't understand speech
  • Needs external processing + maybe internet
  • More coding + data handling
  • More failure points during demo
  • Good as “advanced / future upgrade”
Total ≈ ₹800 – ₹1,000 (excluding laptop/Pi)
Simpler
Option B: Voice Module + ESP32
Speak → Module recognizes ON/OFF → Sends to ESP32 → Light ON/OFF
Brain Location:
Inside the voice module (built-in recognition).
Output:
LED or real bulb with relay.
Parts & Cost (₹)
  • Voice module 1,400 – 2,200
  • ESP32 board 250 – 400
  • Relay (if bulb) 80 – 200
  • LED/Bulb + holder 10 – 200
  • Wires/Breadboard 100 – 200
Key Points
  • Understands trained ON/OFF commands
  • No laptop or internet needed
  • Simple wiring + simple code
  • More reliable in final demo
  • Clean, self-contained setup
Total ≈ ₹2,000 – ₹2,500
